## 1. Broad Area of Work

Conversational agents and chatbots are emerging research areas in the current era of Information Science. The broad scope of this project includes Artificial Intelligence, Information Retrieval, and Information Extraction techniques. The current research project focuses on the following application areas:

- **Natural Language Processing:** To extract the right information  
- **Knowledge Representation:** To store information during interrogation  
- **Automated Reasoning:** To use both new and stored information to draw conclusions  
- **Machine Learning:** To adapt and detect new patterns  

---

## 2. Background

Technologies are becoming increasingly intelligent and are now an essential part of human life. In today’s era, people are highly dependent on web search engines to seek information. However, searching for information on the web can be stressful and increase cognitive load.

A smart search agent is therefore in high demand. A conversational agent (intelligent agent) can act as an intermediary between the user and the web search engine, helping users retrieve desired information more effectively while reducing cognitive load.

---

## 3. Objectives

The objectives of this project are:

- To understand the current scenario of chatbots and the need for smart search  
- To design an agent that acts as a human intermediary between users and web search engines  
- To build a conversational search agent to assist users in searching queries on the web  
- To validate the prototype through user evaluation  

---

## 4. Scope of Work

The scope of this dissertation is to design a dialogue-based conversational search agent that processes user queries through a web search engine and delivers relevant information.
